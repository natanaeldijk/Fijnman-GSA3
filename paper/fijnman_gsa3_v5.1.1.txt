\large{(v5.1.1: Layered Symbolic Core with Constitutional Grounding)}}
\author{Natana\"el van Dijk}

\begin{abstract}
Contemporary reasoning systems exhibit persistent instability: identical inputs can yield multiple, mutually incompatible outputs depending on interpretation, hidden assumptions, or implicit completion of missing information. This instability is not incidental but structural: it arises from permitting inference under under‑specification.

We present a fail‑closed framework for reasoning in which admissibility replaces truth as the primary criterion for existence. A statement may exist only if it is fully specified and cannot be structurally invalidated by alternative interpretations. Any attempt to infer beyond explicitly defined variables and relations results in a mandatory halt.

In this version (v5.1.1), we extend the framework with a layered symbolic core grounded in a constitutional quotient. State spaces are represented symbolically via constraints, and invariants are organised into a hierarchy \(\Phi_0 \subseteq \Phi_1 \subseteq \dots \subseteq \Phi_k\). Knowledge is defined level‑dependently as the maximal \(\Phi_i\)-consistent residue of a successor set. This structure avoids enumerative explosions, handles uncertainty without probability, and preserves fail‑closed semantics. A mode profile replaces the single mode classification, and admissibility is determined by the highest non‑empty kernel level \(I^*\). The constitutional quotient \(\hat{S} = S/{\equiv_{\Phi_0}}\) provides a clean topological foundation.

All extensions preserve the fail‑closed nature of the framework. The categorical analysis of reachability (v4.9) remains as a complementary layer, motivating enriched morphisms without altering the core validation pipeline.
\end{abstract}

\section{Introduction}
\label{sec:intro}

Reasoning systems—both human and computational—routinely produce divergent outputs from identical inputs. Given a dataset \(D\) and a method \(M\), it is common that
\[
M_1(D)\to O_1,\qquad M_2(D)\to O_2,\quad\text{with }O_1\neq O_2.
\]
Such divergence is typically attributed to differences in models, perspectives, or interpretation. However, this explanation is incomplete. The deeper issue is that most systems permit inference under under‑specified conditions.

When essential variables, constraints, or relations are not explicitly defined, systems compensate by introducing implicit assumptions. These assumptions may remain unexamined, yet they directly influence outcomes. As a result, reasoning becomes dependent on hidden completions rather than explicit structure.

This paper adopts a different approach. Instead of attempting to improve inference under under‑specification, we prohibit it.

We construct a fail‑closed framework in which reasoning is only permitted when all required components are explicitly specified. Missing definitions, ambiguous mappings, or competing interpretations are not resolved through approximation or estimation, but trigger a mandatory halt condition.

Within this framework, existence is redefined: a statement does not exist because it is asserted or interpretable, but because it is admissible under a fully specified structure. Admissibility replaces truth as the primary criterion. A statement is admissible if and only if it can be evaluated without introducing new assumptions.

Reasoning is modeled as a sequence of admissible state transitions within a constrained system. Each transition must preserve a set of invariants; any violation results in rejection. Knowledge is not defined as belief or justified truth, but as the set of statements that remain invariant across all admissible continuations.

A central component of the framework is a convergence requirement: iterative refinement must approach a stable limit without the introduction of new assumptions. If refinement requires additional assumptions, the process diverges and is terminated. This establishes a structural boundary against inference under under‑specification.

\subsection{Contribution of Version 5.1.1}

Version 5.1.1 replaces the enumerative state space and flat invariant set with a \emph{layered symbolic core} and introduces a \emph{constitutional quotient} that separates structural grounding from epistemic refinement:
\begin{itemize}
  \item \textbf{Symbolic state space}: \(S = \{\sigma \mid \Psi(\sigma)\}\) where \(\Psi\) is a logical formula; states are assignments to variables.
  \item \textbf{Constitutional quotient}: \(\hat{S} = S/{\equiv_{\Phi_0}}\) where \(\Phi_0\) are minimal structural invariants. All topological and dynamic operations are defined on \(\hat{S}\).
  \item \textbf{Invariant hierarchy}: \(\Phi_0 \subseteq \Phi_1 \subseteq \dots \subseteq \Phi_k\), with \(\Phi_0\) constitutional and \(\Phi_i\) (\(i>0\)) epistemic refinements.
  \item \textbf{Level‑relative kernel}: \(K^{(i)}(S') = S'\) if \(S'\) is \(\Phi_i\)-consistent, otherwise \(\emptyset\). The kernel is the maximal \(\Phi_i\)-consistent \emph{residue} of \(S'\). This definition deliberately avoids selecting maximal consistent subsets; any internal inconsistency collapses the entire set.
  \item \textbf{Admissibility}: The highest level \(I^* = \max\{i \mid K^{(i)}(S') \neq \emptyset\}\) determines the outcome.
  \item \textbf{Mode profile}: A vector of modes per level, replacing the single mode classification.
\end{itemize}
These extensions preserve fail‑closed semantics while enhancing scalability and handling uncertainty without probability. The categorical analysis of reachability (v4.9) remains as a complementary layer, motivating enriched morphisms without altering the core validation pipeline.

\section{Constitutional Definitions}
\label{sec:constitution}

The framework is built on a layered symbolic core. All definitions are mechanistically executable: every step reduces to formula manipulation or solver invocation, and all choices are deterministic.

\subsection{Symbolic State Space}
A state space is defined by a formula \(\Psi\) in a decidable logic (e.g., QF\_LIA, QF\_UF):
\[
S = \{\sigma \mid \Psi(\sigma)\},
\]
where \(\sigma\) ranges over assignments to variables.  
\(\Psi\) must be fully specified. If variables are missing, constraints are ambiguous, or the logic falls outside the specified class:
\[
\mapsto \textsc{Halt\_Spec\_Required}.
\]

\subsection{Constitutional Quotient}
All topological and dynamic operations are defined on the quotient space induced by the constitutional invariants \(\Phi_0\):
\[
\hat{S} = S / \equiv_{\Phi_0},
\]
where \(\equiv_{\Phi_0}\) identifies states with identical \(\Phi_0\)-values.  
Symbolic state definitions and successor constraints are interpreted in \(\hat{S}\) after quotienting; the quotient map is denoted \(\pi : S \to \hat{S}\).

\subsection{Invariant Hierarchy}
A finite, totally ordered family of invariants refines the constitutional base:
\[
\Phi_0 \subseteq \Phi_1 \subseteq \dots \subseteq \Phi_k.
\]
\(\Phi_0\) are the minimal, always‑required invariants (the constitutional layer).  
For \(i>0\), \(\Phi_i\) are stricter epistemic refinement layers (finer distinctions).

\subsection{Symbolic Substate}
A successor set \(S' \subseteq \hat{S}\) is represented symbolically as:
\[
S' = \{\sigma \mid \Psi(\sigma) \land \Xi(\sigma)\},
\]
where \(\Xi\) is an additional constraint in the same logic as \(\Psi\).  
If \(\Psi \land \Xi\) is unsatisfiable, then \(S' = \emptyset\).

\subsection{\(\Phi_i\)-Equivalence and Consistency}
For level \(i\):
\[
\sigma \sim_{\Phi_i} \sigma' \iff \forall \phi \in \Phi_i : \phi(\sigma) = \phi(\sigma').
\]
The equivalence class of \(\sigma\) is the logical formula:
\[
[\sigma]_{\Phi_i} = \{ \tau \mid \Psi(\tau) \land \bigwedge_{\phi \in \Phi_i} \phi(\tau) = \phi(\sigma) \}.
\]
\(S'\) is \emph{\(\Phi_i\)-consistent} if:
\[
\forall \sigma_1,\sigma_2 \in S' : \forall \phi \in \Phi_i : \phi(\sigma_1) = \phi(\sigma_2).
\]
Symbolically, this is equivalent to:
\[
\forall \tau_1,\tau_2 : (\Psi(\tau_1) \land \Xi(\tau_1) \land \Psi(\tau_2) \land \Xi(\tau_2)) \Rightarrow \bigwedge_{\phi \in \Phi_i} (\phi(\tau_1) = \phi(\tau_2)).
\]
If this formula is valid in the chosen logic, \(S'\) is \(\Phi_i\)-consistent.

\subsection{Level‑Relative Kernel}
\[
K^{(i)}(S') = \begin{cases}
S' & \text{if } S' \text{ is } \Phi_i\text{-consistent},\\
\emptyset & \text{otherwise}.
\end{cases}
\]
\emph{Interpretation.} The kernel is the maximal \(\Phi_i\)-consistent \emph{residue} of \(S'\): when \(S'\) is consistent it coincides with \(S'\) itself; otherwise it is empty, indicating that no common \(\Phi_i\)-value exists across all \(\sigma \in S'\). This definition deliberately avoids selecting maximal consistent subsets; any internal inconsistency collapses the entire set.

\subsection{Kernel Monotonicity}
\begin{theorem}[Kernel Monotonicity]
For all \(i < j\):
\[
K^{(j)}(S') \subseteq K^{(i)}(S').
\]
\end{theorem}
\begin{proof}
Because \(\Phi_i \subseteq \Phi_j\), \(\Phi_j\)-consistency implies \(\Phi_i\)-consistency. Two cases:
\begin{itemize}
  \item If \(S'\) is \(\Phi_j\)-consistent, then it is also \(\Phi_i\)-consistent, so \(K^{(j)}(S') = S' = K^{(i)}(S')\).
  \item If \(S'\) is not \(\Phi_j\)-consistent, then \(K^{(j)}(S') = \emptyset \subseteq K^{(i)}(S')\).
\end{itemize}
Thus the inclusion holds in all cases.
\end{proof}

\subsection{Admissibility Criterion (\(I^*\))}
\[
I^* = \max\{\, i \mid K^{(i)}(S') \neq \emptyset \,\}.
\]
If \(I^*\) exists:
\[
\mapsto \textsc{Admissible\_Level}(I^*).
\]
If no level yields a non‑empty kernel:
\[
\mapsto \textsc{Refusal}.
\]
The index set is finite and totally ordered, so \(I^*\) is well‑defined (or absent).

\subsection{Knowledge}
\[
\text{Knowledge}_i(S') = K^{(i)}(S').
\]
Knowledge is level‑dependent; the system reports the highest level with non‑empty residue.

\subsection{Modes per Level}
For each level \(i\), given \(S' = \{\sigma \mid \Psi(\sigma) \land \Xi(\sigma)\}\):
\begin{itemize}
  \item \textsc{Strict}: exactly one model of \(\Psi \land \Xi\) exists, and uniqueness is decidable in the chosen logic. Uniqueness is checked via solver‑based disequality test: \(\exists! \sigma : \Psi(\sigma)\land\Xi(\sigma)\) is equivalent to \(\forall \sigma_1,\sigma_2 : (\Psi(\sigma_1)\land\Xi(\sigma_1)\land\Psi(\sigma_2)\land\Xi(\sigma_2)) \Rightarrow \sigma_1=\sigma_2\).
  \item \textsc{Safe}: at least two distinct models exist, and \(S'\) is \(\Phi_i\)-consistent.
  \item \textsc{Defeasible}: \(S'\) is not \(\Phi_i\)-consistent.
\end{itemize}
These cases are mutually exclusive and tested in the listed order.  
If uniqueness cannot be determined (solver returns \textsc{Unknown} or the theory does not support uniqueness checking):
\[
\mapsto \textsc{Refusal}.
\]

\subsection{Mode Profile}
\[
\text{ModeProfile}(S') = (m_0, m_1, \dots, m_k),
\]
where each \(m_i\) is the mode of level \(i\).  
The profile is descriptive only; admissibility is determined solely by \(I^*\).

\subsection{Dynamic Updates}
An update \(u\) selects a surviving subset \(\hat{S}_u \subseteq \hat{S}\) (the quotient space). The reachability order is restricted accordingly:
\[
R_A^u = R_A \cap (\hat{S}_u \times \hat{S}_u),\qquad \leq_u = (R_A^u)^*.
\]
If \(\hat{S}_u = \emptyset\), the system halts with \textsc{Refusal}, preserving fail‑closed semantics.

For any successor set \(S' \subseteq \hat{S}_u\), the kernel after update is:
\[
K^{(i),u}(S') = K^{(i)}(S') \cap \hat{S}_u,
\]
where the intersection is taken after first restricting \(S'\) to \(\hat{S}_u\) (i.e., \(S'\) is already understood as a subset of \(\hat{S}_u\)).

\subsection{Decidability and Solver Behavior}
\[
\begin{array}{|l|l|}
\hline
\text{Situation} & \text{Outcome} \\
\hline
\text{Logic outside specified class} & \textsc{Halt\_Spec\_Required} \\
\text{Solver returns \textsc{Unknown}} & \textsc{Refusal}\quad (\text{treated as semantic non-admissibility}) \\
\text{Query well‑formed but not decidable by the adopted procedure} & \textsc{Refusal} \\
\hline
\end{array}
\]

\subsection{Axioms}
\label{sec:axioms}
The following axioms govern the transition structure.

\begin{axiom}[Invariant Preservation]
For every admissible transition \((\sigma,\tau)\in R_A\) and every invariant \(\phi\in\Phi_0\) (the constitutional invariants), we have \(\phi(\sigma)=\phi(\tau)\). Invariants at higher levels are preserved by definition of consistency.
\end{axiom}

\begin{axiom}[Separation]
If \(\sigma \leq \tau\) and \(\tau \leq \sigma\) (where \(\leq\) is the reflexive‑transitive closure of \(R_A\)), then \([\sigma]_{\Phi_0} = [\tau]_{\Phi_0}\). In the quotient \(\hat{S}=S/{\equiv_{\Phi_0}}\), this ensures antisymmetry of the reachability order.
\end{axiom}

\section{Relational Admissibility: RGR and RCR}
\label{sec:admissibility}

The relational admissibility gates are applied sequentially as in v4.9.1, now interpreted in the symbolic setting and referring to the constitutional invariants \(\Phi_0\).

\subsection{Relational Grounding Requirement (RGR)}
A state \(\sigma\) is relationally grounded iff its relational component \(R_\sigma \neq \emptyset\). Inference is admissible only over relationally grounded states. Formally, the state space formula must satisfy:
\[
\Psi(\sigma) \Rightarrow R_\sigma \neq \emptyset.
\]

At the proposal level, failure of relational grounding yields:
\[
R = \emptyset \Rightarrow A_{\text{pre}}(q) = \textsc{Halt\_Spec\_Required}.
\]
At the state level, any attempted representation with \(R_\sigma = \emptyset\) fails completeness and therefore does not belong to \(S\).

\subsection{Relational Closure Requirement (RCR)}
A relationally grounded state \(\sigma\) satisfies the Relational Closure Requirement iff its relational structure is consistent with the minimal invariant family \(\Phi_0\):
\[
\Delta\Phi_0(\sigma) = 0.
\]
A state that is grounded but fails closure is classified as inconsistent:
\[
R_\sigma \neq \emptyset \land \Delta\Phi_0(\sigma) \neq 0 \Rightarrow A(q, \sigma) = \textsc{Reject}.
\]

\subsection{Admissibility Pipeline}
For a proposal \(q\) and a state \(\sigma\) that results from mapping \(q\) to \(\sigma\) (when possible), the admissibility outcome is determined as follows:
\begin{itemize}
  \item \(A_{\text{pre}}(q) = \textsc{Halt\_Spec\_Required}\) ⇒ no state is formed; system halts.
  \item Otherwise, \(R_\sigma \neq \emptyset\), \(\Delta\Phi_0(\sigma) \neq 0\) ⇒ \(A(q, \sigma) = \textsc{Reject}\).
  \item Otherwise, \(R_\sigma \neq \emptyset\), \(\Delta\Phi_0(\sigma) = 0\) ⇒ \(A(q, \sigma) = \textsc{Admissible}\) (proceed to transition).
\end{itemize}
Grounding permits evaluation. Closure permits admissibility. The kernel is what survives both.

\section{Dynamics and Knowledge}
\label{sec:knowledge}

\subsection{Trajectory}
\[
\sigma_0 \rightarrow \sigma_1 \rightarrow \sigma_2 \rightarrow \dots
\]

\subsection{Properties}
\begin{itemize}
  \item \textbf{Linearity}: no inadmissible return to a prior state without new admissible information.
  \item \textbf{Convergence}: \(\lim_{n\to\infty} \sigma_n = \sigma^*\) (if a unique fixed point exists).
  \item \textbf{Divergence}: if no stable kernel exists, \(\to\) \textsc{Refusal}.
\end{itemize}

\subsection{Knowledge Object}
A set \(K \subseteq S\) is forward‑closed under admissible transitions if
\[
\forall \sigma \in K, \forall q \text{ admissible}: \Theta(q, \sigma) \subseteq K.
\]

\subsection{Knowledge as Level‑Relative Kernel}
Knowledge is not a single set but a family indexed by invariant level:
\[
\text{Knowledge}_i(S') = K^{(i)}(S').
\]
Knowledge is level‑dependent; there is no global truth, only structural invariant residue. The system reports the highest level \(I^*\) at which a non‑empty kernel exists.

  \item \textsc{Safe}: there exist at least two distinct models of \(\Psi \land \Xi\), and \(S'\) is \(\Phi_i\)-consistent.
  \item \textsc{Defeasible}: \(S'\) is not \(\Phi_i\)-consistent.
\end{itemize}
These cases are mutually exclusive and tested in the order listed.

\paragraph{Decidability for Strict.}
If uniqueness cannot be determined (solver returns \textsc{Unknown}, or the theory does not support uniqueness checking):
\[
\mapsto \textsc{Refusal}.
\]

\subsection{Mode Profile}
\[
\text{ModeProfile}(S') = (m_0, m_1, \dots, m_k)
\]
where each \(m_i\) is the mode of level \(i\). The profile fully describes the system state.

\paragraph{Descriptive status.}
\(\text{ModeProfile}\) is descriptive only and does not alter admissibility, which is determined exclusively by \(I^* = \max\{i \mid K^{(i)}(S') \neq \emptyset\}\).

\section{Multi-Agent Extension}
\label{sec:multiagent}

\emph{This section remains largely unchanged from v4.9.1, with the adaptation that each agent’s state space is symbolic and invariants may be layered. The joint kernel is defined as the intersection of per‑agent kernels over all levels.}

\subsection{Agent Definition}
An agent \(a \in \mathcal{A}\) has:
\begin{itemize}
  \item Individual state space \(S_a \subseteq S\) (symbolic)
  \item Proposal set \(Q_a \subseteq Q\)
  \item Admissibility operator \(A_a : Q_a \times S_a \to \Omega\)
  \item Guard \(G_a : S_a \times Q_a \to \{0,1\}\)
  \item Individual invariants \(\Phi_a\) (a hierarchy)
\end{itemize}

\subsection{Joint State and Proposals}
Joint state:
\[
\Sigma = ((S_a)_{a\in\mathcal{A}}, C, \mathcal{H})
\]
Joint proposal:
\[
\mathbf{q} = (q_1,\ldots,q_n) \in Q_{\text{joint}} = \prod_{a\in\mathcal{A}} Q_a
\]
Joint admissibility:
\[
A_{\text{joint}}(\mathbf{q}, \Sigma) = \textsc{Admissible} \iff \forall a : A_a(q_a, S_a) = \textsc{Admissible}
\]
Joint guard:
\[
G_{\text{joint}}(\Sigma, \mathbf{q}) = 1 \iff \forall a : G_a(S_a, q_a) = 1
\]

\subsection{Multi-Agent Invariants}
\[
\Phi_{\text{joint}} = \bigcup_a \Phi_a \cup \{\phi_{\text{comm}}, \phi_{\text{cons}}, \phi_{\text{priv}}\}
\]
\begin{itemize}
  \item \(\phi_{\text{comm}}\): communication is explicit, traceable, and admissible
  \item \(\phi_{\text{cons}}\): shared facts are consistent across all agents
  \item \(\phi_{\text{priv}}\): private information is shared only with explicit consent
\end{itemize}

\subsection{Joint Kernel}
\[
K_{\text{safe,joint}}(\mathbf{q}, \Sigma) = \bigcap_{a\in\mathcal{A}} K_{\text{safe},a}(S_a')
\]
The joint kernel is level‑wise, and \(I^*\) is computed as the maximum level where all agents have non‑empty kernels.

\section{Relation to Epistemic Logic}
\label{sec:epistemic}

\emph{This section is unchanged from v4.9.1; it highlights the conceptual shift from possibility to admissibility.}

\subsection{Common Ground}
Both epistemic logic and this framework are concerned with knowledge, information, and dynamics under updates or transitions.

\subsection{Key Differences}
\paragraph{Possibility vs Admissibility}
Classical epistemic logic asks: what is possible? Knowledge is defined as truth across all accessible possible worlds. This framework asks: what is admissible? Knowledge is invariance under admissible transitions.

\paragraph{Updates vs Validation}
Dynamic epistemic logic models knowledge updates as operations on epistemic models. This framework separates generation from admissibility: proposals are validated against explicit invariants.

\paragraph{Constraints vs Possible Worlds}
Epistemic logic works with possible worlds and accessibility relations. This framework works with explicit invariants \(\Phi\) and admissible transitions \(\Theta\) (equivalently, the induced relation \(R_A\)).

\paragraph{Common Knowledge as Fixed Point}
In dynamic epistemic logic, common knowledge is the fixed point of mutual knowledge. In this framework, common knowledge corresponds to the fixed point of the joint kernel:
\[
K_{\text{safe,joint}} = \bigcap_a K_{\text{safe},a}.
\]

\subsection{Mapping (Not Identity)}
\begin{itemize}
  \item Kripke worlds \(\rightarrow\) states \(S\)
  \item Accessibility relations \(\rightarrow\) admissible transitions \(R_A\)
  \item Common knowledge \(\rightarrow\) joint kernel fixed point
\end{itemize}

\subsection{Why the Shift}
The shift from possibility to admissibility is motivated by applications in which underspecification must be handled explicitly rather than silently completed.

\section{Scope and Limitations}
\label{sec:limitations}

The framework presented here is a structural validation system, not a general reasoning engine.

\subsection{What the System Does}
The system determines whether a proposal is admissible given:
\begin{itemize}
  \item a fully specified symbolic state space \(\Psi\)
  \item a complete set of invariant hierarchies \(\Phi_i\)
  \item a well‑defined transition operator \(\Theta\) (generating symbolic \(S'\))
\end{itemize}
It validates, filters, and executes admissible transitions. It does not generate proposals, infer missing specifications, or assign probabilities.

\subsection{What the System Does Not Do}
\begin{itemize}
  \item Generate content: proposals must be supplied externally.
  \item Handle probability: uncertainty is modeled via kernel emptiness at higher levels, not distributions.
  \item Seek completeness: if specification is missing, the system halts.
  \item Optimize: there is no utility function and no preference ordering.
  \item Explain: the system does not provide causal or intentional explanations.
\end{itemize}

\subsection{Boundaries}
The system is restricted to validation, while generation, specification, and interpretation remain external. All content, including \(\Psi\), \(\Phi_i\), and proposals, is supplied externally. The system only checks consistency, completeness, and convergence.

\subsection{Applicability}
The framework is suited for domains where:
\begin{itemize}
  \item structural constraints dominate over intentions or narratives
  \item admissibility is more relevant than possibility
  \item underspecification must be handled by halting, not guessing
\end{itemize}

\subsection{Limitations}
The framework does not claim to:
\begin{itemize}
  \item replace existing logics (DEL, HoTT, etc.)
  \item solve all reasoning problems
  \item guarantee correctness of external inputs
  \item provide a complete theory of knowledge
\end{itemize}
Its strength lies in structural discipline, not in expressive richness. It is a filter, not a generator.

\section{Topological Grounding of Admissible Reasoning}
\label{sec:topology}

The definitions of \(S\) (symbolic), \(R_A\), \(\Phi_i\), and the kernel admit a natural topological interpretation. All states in \(S\) are relationally grounded (\(R_\sigma \neq \emptyset\)), so the space is free of pre‑state failures.

We work on the quotient space induced by the constitutional invariants \(\Phi_0\):
\[
\hat{S} = S / \equiv_{\Phi_0}
\]
where \(\equiv_{\Phi_0}\) identifies states with identical \(\Phi_0\)-values. The reachability order descends to a partial order on \(\hat{S}\). We denote the quotient map by \(\pi : S \to \hat{S}\). All topological notions are defined on \(\hat{S}\). Higher invariant levels \(\Phi_i\) (\(i>0\)) refine this quotient but do not change the underlying topology.

\subsection{Poset from Reachability}
Let \(\leq\) be the reflexive‑transitive closure of \(R_A\) on \(\hat{S}\). Because cycles without new admissible information are inadmissible and Axiom~2 ensures such cycles collapse to the same \(\Phi_0\)-class, \(\leq\) is antisymmetric on \(\hat{S}\). Hence \((\hat{S}, \leq)\) is a partially ordered set (poset).

\subsection{Forward‑Stable Sets and Topology}
A set \(U \subseteq \hat{S}\) is forward‑stable under \(R_A\) iff it is upward‑closed under \(\leq\). The family \(\mathcal{T}\) of all upward‑closed subsets of \((\hat{S}, \leq)\) is an Alexandrov topology. We call \(\mathcal{T}\) the admissible topology on \(\hat{S}\).

\subsection{Kernel Openness}
For any non‑empty successor set \(S' \subseteq \hat{S}\), the base kernel can be expressed topologically as the intersection of \(\Phi_0\)-classes:
\[
K^{(0)}(S') = \bigcap_{\sigma' \in S'} [\sigma']_{\Phi_0}.
\]
Operationally, v5.1.1 retains the consistency‑based kernel definition of Section~\ref{sec:constitution}, which for \(\Phi_0\) coincides with this topological representation. Since each \([\sigma']_{\Phi_0}\) is upward‑closed, and arbitrary intersections of upward‑closed sets are upward‑closed, \(K^{(0)}(S')\) is open in \(\mathcal{T}\).

\subsection{Mode Refinement via Topology}
The mode conditions of Section~\ref{sec:modes} can be understood topologically:
\begin{itemize}
  \item \textsc{Strict}: \(|S'| = 1\) implies the kernel is a single \(\Phi_0\)-class, which is open.
  \item \textsc{Safe}: all successors lie in the same \(\Phi_0\)-class, so the kernel is that non‑empty open class.
  \item \textsc{Defeasible}: successors belong to distinct \(\Phi_0\)-classes; their intersection is empty, hence the kernel is \(\emptyset\).
\end{itemize}
This topological perspective shows that the empty kernel in defeasible cases is not an error but a structural necessity: the system correctly refuses to posit common knowledge when admissible branches diverge in invariant terms.

\section{Dynamic Updates as Topological Refinement}
\label{sec:updates}

\subsection{Update as Restriction}
An update \(u\) selects a subset \(\hat{S}_u \subseteq \hat{S}\) of states that survive. The new reachability relation is the restriction:
\[
R_A^u = R_A \cap (\hat{S}_u \times \hat{S}_u).
\]
The new poset order \(\leq_u\) is the reflexive‑transitive closure of \(R_A^u\). If \(\hat{S}_u = \emptyset\), the system halts with \textsc{Refusal}, preserving fail‑closed.

\subsection{Topological Interpretation}
The admissible topology \(\mathcal{T}\) consists of all upward‑closed subsets of \((\hat{S}, \leq)\). For any open set \(U \in \mathcal{T}\), \(U\) can be interpreted as the set of states compatible with current knowledge. The update restricts the state space to \(\hat{S}_u\); the new topology \(\mathcal{T}_u\) is the Alexandrov topology induced by \(\leq_u\). The restriction operator
\[
\operatorname{int}_u : \mathcal{T} \to \mathcal{T}_u,\quad \operatorname{int}_u(U) = U \cap \hat{S}_u
\]
is decreasing, monotone, and idempotent.

\subsection{Effect on Equivalences}
After an update, both the set of states and the reachability structure are restricted. Consequently, the equivalence relations are recomputed on the new poset:
\[
\sigma \sim_{\Phi_0}^u \tau \iff [\sigma]_{\Phi_0}^u = [\tau]_{\Phi_0}^u,\qquad
\sigma \sim_Y^u \tau \iff \sigma \leq_u \tau \text{ and } \tau \leq_u \sigma.
\]
Because \(\hat{S}_u \subseteq \hat{S}\) and \(\leq_u \subseteq \leq\), equivalence classes can only become smaller or disappear entirely. Thus updates make knowledge weakly sharper: they never coarsen distinctions.

\subsection{Kernel and Modes After an Update}
The kernel is recomputed on the new successor set. For a transition step with immediate successors \(S' \subseteq \hat{S}_u\), we have
\[
K^{(i),u}(S') = K^{(i)}(S') \cap \hat{S}_u,
\]
where the intersection is taken after first restricting \(S'\) to \(\hat{S}_u\) (i.e., \(S'\) is already understood as a subset of \(\hat{S}_u\)). Because the \(\Phi_i\)-classes are open in \(\mathcal{T}_u\), the kernel remains open in the refined topology. The mode classification is applied exactly as before, now relative to \(\leq_u\) and \(\sim_{\Phi_i}^u\).

\subsection{Relation to Dynamic Epistemic Logic}
In DEL, a public announcement \(\psi\) eliminates all worlds where \(\psi\) is false, which corresponds directly to a restriction \(\hat{S}_u = \{\sigma \in \hat{S} \mid \sigma \models \psi\}\). The structural core of an update is therefore identical. While DEL is formulated over Kripke models, our framework operates on a poset induced by admissible reachability, suggesting a natural translation for acyclic, antisymmetric models.

\section{Formal Properties: Lemmas and Theorems}
\label{sec:theorems}

We now consolidate the framework into a formal theory by stating and proving the key properties established in the preceding sections.

\subsection{Order and Topology}

\begin{lemma}[Poset Structure]
The reachability order \(\leq\) defined as the reflexive‑transitive closure of \(R_A\) on the quotient \(\hat{S}\) is a partial order: reflexive, transitive, and antisymmetric.
\end{lemma}
\begin{proof}
Reflexivity and transitivity hold by construction. Antisymmetry follows from Axiom~2: if \(\sigma \leq \tau\) and \(\tau \leq \sigma\) then \([\sigma]_{\Phi_0} = [\tau]_{\Phi_0}\). In the quotient this forces \(\sigma = \tau\).
\end{proof}

\begin{lemma}[Forward‑stable = Upward‑closed]
A set \(U \subseteq \hat{S}\) is forward‑stable under \(R_A\) iff it is upward‑closed under \(\leq\).
\end{lemma}
\begin{proof}
Forward‑stability gives closure under single \(R_A\)-steps; by induction it extends to all finite paths, hence to \(\leq\). Conversely, upward‑closed under \(\leq\) immediately implies closure under \(R_A\) because \(R_A \subseteq \leq\).
\end{proof}

\begin{theorem}[Openness of \(\Phi_i\)-Classes in \(S\)]
For any \(\sigma \in S\), the set \([\sigma]_{\Phi_i} \subseteq S\) is open in the Alexandrov topology induced by \(\leq\) on \(S\).
\end{theorem}
\begin{proof}
Take any \(\sigma' \in [\sigma]_{\Phi_i}\) and any \(\tau\) with \(\sigma' \leq \tau\). By Axiom~1 and the definition of \(\leq\), \(\phi(\tau) = \phi(\sigma')\) for all \(\phi \in \Phi_i\), so \(\tau \in [\sigma]_{\Phi_i}\). Hence \([\sigma]_{\Phi_i}\) is upward‑closed, and therefore open.
\end{proof}

\begin{theorem}[Kernel Openness]
For any non‑empty successor set \(S' \subseteq \hat{S}\), the kernel \(K^{(0)}(S') = \bigcap_{\sigma' \in S'} [\sigma']_{\Phi_0}\) is open in \(\mathcal{T}\).
\end{theorem}
\begin{proof}
Each \([\sigma']_{\Phi_0}\) is upward‑closed, and arbitrary intersections of upward‑closed sets are upward‑closed. Hence \(K^{(0)}(S')\) is open.
\end{proof}

\subsection{Admissibility Pipeline}

\begin{theorem}[RGR Failure]
If a proposal \(q\) has empty relational component (\(R = \emptyset\)), then \(A_{\text{pre}}(q) = \textsc{Halt\_Spec\_Required}\).
\end{theorem}
\begin{proof}
The mapping \(M : Q \to S \cup \{\bot\}\) preserves the relational component, so if \(R = \emptyset\) the resulting tuple would have \(R_\sigma = \emptyset\). By the completeness axiom, \(\neg \text{Complete}(\sigma)\). Hence the state cannot be formed, and the pre‑state check returns \textsc{Halt\_Spec\_Required}.
\end{proof}

\begin{theorem}[RCR Failure]
If a state \(\sigma\) is relationally grounded (\(R_\sigma \neq \emptyset\)) but fails the invariant condition (\(\Delta\Phi_0(\sigma) \neq 0\)), then \(A(q, \sigma) = \textsc{Reject}\).
\end{theorem}
\begin{proof}
Directly from the definition of the Relational Closure Requirement.
\end{proof}

\begin{theorem}[Pipeline Exhaustiveness]
For any admissible proposal \(q\) and any state \(\sigma\) resulting from the mapping \(M(q)\), exactly one of the following holds:
\begin{itemize}
  \item \(A_{\text{pre}}(q) = \textsc{Halt\_Spec\_Required}\) (RGR failure),
  \item \(A(q, \sigma) = \textsc{Reject}\) (RCR failure),
  \item \(A(q, \sigma) = \textsc{Admissible}\) (both gates satisfied).
\end{itemize}
Thus the admissibility pipeline is exhaustive over the relevant cases.
\end{theorem}
\begin{proof}
The three cases are mutually exclusive by construction and cover all possibilities because either the proposal cannot be mapped to a complete state (RGR failure) or it can; in the latter case, either the invariants hold (RCR success) or they do not (RCR failure).
\end{proof}

\subsection{Kernel and Mode Behaviour}

\begin{theorem}[Kernel Monotonicity]
For all \(i < j\):
\[
K^{(j)}(S') \subseteq K^{(i)}(S').
\]
\end{theorem}
\begin{proof}
Because \(\Phi_i \subseteq \Phi_j\), \(\Phi_j\)-consistency implies \(\Phi_i\)-consistency. Two cases:
\begin{itemize}
  \item If \(S'\) is \(\Phi_j\)-consistent, then it is also \(\Phi_i\)-consistent, so \(K^{(j)}(S') = S' = K^{(i)}(S')\).
  \item If \(S'\) is not \(\Phi_j\)-consistent, then \(K^{(j)}(S') = \emptyset \subseteq K^{(i)}(S')\).
\end{itemize}
Thus the inclusion holds in all cases.
\end{proof}

\begin{theorem}[Strict Kernel]
In Strict mode, \(|S'| = 1\) and \(S'\) is \(\Phi_i\)-consistent for all \(i\). Hence \(K^{(i)}(S') = S'\) for all \(i\).
\end{theorem}
\begin{proof}
With a unique model, all \(\Phi_i\)-values are trivially constant across \(S'\). Thus \(S'\) is \(\Phi_i\)-consistent for every \(i\).
\end{proof}

\begin{theorem}[Safe Kernel]
In Safe mode, \(|S'| \ge 2\) and \(S'\) is \(\Phi_i\)-consistent. Hence \(K^{(i)}(S') = S'\).
\end{theorem}
\begin{proof}
Immediate from the definition of Safe and the kernel definition.
\end{proof}

\begin{theorem}[Defeasible Kernel]
In Defeasible mode, \(S'\) is not \(\Phi_i\)-consistent, so \(K^{(i)}(S') = \emptyset\).
\end{theorem}
\begin{proof}
Immediate from the definition of Defeasible and the kernel definition.
\end{proof}

\subsection{Updates and Convergence}

\begin{lemma}[Restricted \(\Phi_i\)-classes]
For an update \(u\) with surviving state space \(\hat{S}_u \subseteq \hat{S}\) and any \(\sigma \in \hat{S}_u\):
\[
[\sigma]_{\Phi_i}^u = [\sigma]_{\Phi_i} \cap \hat{S}_u.
\]
\end{lemma}
\begin{proof}
By definition, \([\sigma]_{\Phi_i}^u\) consists of those states \(\tau \in \hat{S}_u\) that share all \(\Phi_i\)-values with \(\sigma\). This is exactly the intersection of the original class \([\sigma]_{\Phi_i}\) with \(\hat{S}_u\).
\end{proof}

\begin{theorem}[Kernel under Update]
For an update \(u\) and any successor set \(S' \subseteq \hat{S}_u\):
\[
K^{(i),u}(S') = K^{(i)}(S') \cap \hat{S}_u.
\]
\end{theorem}
\begin{proof}
\[
K^{(i),u}(S') = \bigcap_{\sigma' \in S'} [\sigma']_{\Phi_i}^u = \bigcap_{\sigma' \in S'} ([\sigma']_{\Phi_i} \cap \hat{S}_u) = \left( \bigcap_{\sigma' \in S'} [\sigma']_{\Phi_i} \right) \cap \hat{S}_u = K^{(i)}(S') \cap \hat{S}_u.
\]
\end{proof}

\begin{theorem}[Fail‑closed Updates]
If an update \(u\) leads to \(\hat{S}_u = \emptyset\), the system halts with \textsc{Refusal}, preserving the fail‑closed principle.
\end{theorem}
\begin{proof}
By definition, a transition that would result in an empty state space is not admissible; the system returns \textsc{Refusal}.
\end{proof}

\begin{theorem}[Continuity of Abstraction]
Let \(f : \hat{S} \to \hat{S}'\) be an order‑preserving map between posets (i.e., \(\sigma \leq \tau \Rightarrow f(\sigma) \leq' f(\tau)\)) that respects the invariants: \(\phi_i(\sigma) = \phi_i'(f(\sigma))\) for a corresponding invariant family \(\Phi'\). Then \(f\) is continuous with respect to the Alexandrov topologies.
\end{theorem}
\begin{proof}
Take any open \(U' \subseteq \hat{S}'\) (upward‑closed in \(\leq'\)). We must show \(f^{-1}(U')\) is upward‑closed in \(\leq\). Let \(\sigma \in f^{-1}(U')\) and \(\sigma \leq \tau\). Then \(f(\sigma) \leq' f(\tau)\) by order‑preservation, and \(f(\sigma) \in U'\). Since \(U'\) is upward‑closed, \(f(\tau) \in U'\), so \(\tau \in f^{-1}(U')\). Hence \(f^{-1}(U')\) is upward‑closed, i.e., open.
\end{proof}

\begin{theorem}[Convergence under Iterative Updates]
Let \(u_1, u_2, \dots\) be a sequence of updates such that \(\hat{S}_{u_1} \supseteq \hat{S}_{u_2} \supseteq \cdots\) is a descending chain of subsets. If the chain stabilises after finitely many steps (i.e., the poset satisfies the descending chain condition on the subspace topology), then the kernels converge:
\[
\lim_{n\to\infty} K^{(i),u_n}(S_n') = K^{(i),\lim}(S_{\lim}'),
\]
where the limit is taken in the sense of eventual equality. In particular, if the chain is infinite and the descending chain condition holds, the chain must stabilise at a finite stage.
\end{theorem}
\begin{proof}
In a poset with the descending chain condition (DCC) on the underlying set, any descending chain of subsets eventually becomes constant. The kernels are subsets of these spaces, so they also stabilise. The limit kernel is then the kernel computed on the stable subspace.
\end{proof}

\section{Categorical Extension of the Structural Layer}
\label{sec:categorical}

\emph{This section is largely unchanged from v4.9.1; it provides a categorical perspective on reachability and motivates enriched morphisms. The layered symbolic core does not alter this analysis, which remains an orthogonal addition.}

\subsection{Thin Categorical Reading of the Reachability Structure}
The poset \((\hat{S}, \leq)\) defines a thin category \(\mathcal{C}\):
\begin{itemize}
  \item Objects: states \(\sigma \in \hat{S}\)
  \item Morphisms: there is a unique arrow \(\tau \to \sigma\) iff \(\tau \leq \sigma\)
\end{itemize}
The category is thin because between any two objects there is at most one morphism.

The Yoneda embedding \(y : \mathcal{C} \to [\mathcal{C}^{\text{op}}, \mathbf{Set}]\) sends each state \(\sigma\) to the representable presheaf \(y_\sigma = \operatorname{Hom}_{\mathcal{C}}(-, \sigma)\).

\subsection{Collapse of Yoneda Equivalence in the Thin Case}
\begin{proposition}[Yoneda Equivalence Collapses]
In the thin category \(\mathcal{C}\) defined by the poset \((\hat{S}, \leq)\), two states \(\sigma, \tau \in \hat{S}\) satisfy \(y_\sigma \cong y_\tau\) if and only if \(\sigma = \tau\).
\end{proposition}
\begin{proof}
The Yoneda lemma gives \(y_\sigma \cong y_\tau\) iff \(\operatorname{Hom}_{\mathcal{C}}(-, \sigma) \cong \operatorname{Hom}_{\mathcal{C}}(-, \tau)\). In a thin category this isomorphism holds iff the two representable presheaves are equal as sets of objects, which happens precisely when the downward‑closed sets of objects that can reach \(\sigma\) and \(\tau\) coincide. Because the order is antisymmetric, this forces \(\sigma = \tau\).
\end{proof}
Consequently, structural equivalence \(\sim_Y\) as defined via natural isomorphism of representables reduces to identity. Observational equivalence \(\sim_{\Phi_0}\) is coarser; it can identify distinct states that share the same invariant values, even if they are incomparable in the order. This gap is what we earlier called a structural gap.

\subsection{Why Collapse Matters: The Need for Richer Morphisms}
The collapse theorem shows that in the current setting, where morphisms only encode reachability, the Yoneda embedding cannot distinguish states that are not identical. Therefore any non‑trivial notion of semantic pair that goes beyond mere identity must be based on additional structure on morphisms, not on objects alone.

\subsection{Candidate Enriched Framework}
We propose to enrich the thin category \(\mathcal{C}\) by replacing the set of morphisms (which is at most a singleton) with a set that carries additional data. A natural candidate is to let each admissible transition \((\sigma, \tau)\) (i.e., each \(\sigma \leq \tau\)) be decorated with:
\begin{itemize}
  \item \textbf{Provenance}: a trace of the proposals and guards that generated the transition.
  \item \textbf{Admissibility profile}: which invariants were checked and how the mode (Strict, Safe, Defeasible) was determined.
  \item \textbf{Invariant signature}: the vector of \(\Phi_i\)-values of the states involved.
\end{itemize}
Formally, we define an enriched category \(\mathcal{C}_{\text{enr}}\) with the same objects as \(\hat{S}\), but where the hom‑set \(\operatorname{Hom}_{\text{enr}}(\sigma, \tau)\) is the set of all admissible paths from \(\sigma\) to \(\tau\) together with their decorations. Composition is concatenation of paths (with merged decorations). The original thin category is recovered by forgetting the decorations.

\subsection{Semantic Pairs in the Enriched Setting}
In the enriched framework, we can define a semantic pair \((\sigma, \tau)\) as a pair of states that are connected by an invariant‑enriched equivalence. One way to formalise this is to require the existence of enriched morphisms \(f : \sigma \to \tau\) and \(g : \tau \to \sigma\) such that \(f \circ g\) and \(g \circ f\) are equivalent to identity morphisms under some equivalence relation that respects the decorations.

Because the decorations carry detailed information about the transitions, two distinct states may become isomorphic if there exist admissible paths back and forth that are compatible with the invariant structure. For example, in a Safe mode scenario, all successors lie in the same \(\Phi_i\)-class; if they are mutually reachable via admissible paths, they could be considered semantically equivalent under the enriched interpretation, even though they are distinct in the underlying poset.

\subsection{Boundaries and Future Work}
The enriched framework outlined here is deliberately sketched, not fully formalised. Its purpose is to demonstrate that the collapse of Yoneda equivalence in the thin case is not a weakness but a guide: it tells us where additional structure is needed. The following aspects remain for future work:
\begin{itemize}
  \item Exact definition of the decoration algebra and composition laws.
  \item Compatibility with the invariants \(\Phi_i\) and the kernel \(K^{(i)}\).
  \item Proof that enriched equivalence indeed yields a non‑trivial relation that refines observational equivalence.
  \item Application to multi‑agent scenarios where different agents may have different admissible path decorations.
\end{itemize}
Crucially, the kernel \(K^{(i)}\) remains the set‑theoretic consistency‑based object; it is not replaced by a colimit or any categorical construction. The enriched layer is an orthogonal addition that does not alter the core fail‑closed validation pipeline.

\section{Execution Semantics}
\label{sec:execution}

The framework is mechanised using a satisfiability modulo theories (SMT) solver, e.g., Z3. The following steps define the execution flow:

\begin{enumerate}
  \item \textbf{Input}  
    \begin{itemize}
      \item A symbolic state space \(\Psi\) over a set of variables \(V\).
      \item A hierarchy \(\Phi_0 \subseteq \dots \subseteq \Phi_k\) where each \(\Phi_i\) is a set of formulas over \(V\).
      \item A successor constraint \(\Xi\) (from a proposal \(q\)).
    \end{itemize}
  \item \textbf{Constitutional checks}  
    \begin{itemize}
      \item Verify that \(\Psi\) and \(\Xi\) are well‑formed and use only the declared logic.
      \item Check that \(\Psi(\sigma) \Rightarrow R_\sigma \neq \emptyset\) (RGR).
      \item If any check fails → \(\textsc{Halt\_Spec\_Required}\).
    \end{itemize}
  \item \textbf{Consistency check per level}  
    For each level \(i\) from \(k\) down to 0:  
    \begin{itemize}
      \item Determine whether the substate \(S' = \{\sigma\mid \Psi(\sigma)\land\Xi(\sigma)\}\) is \(\Phi_i\)-consistent by testing if there exist two models of \(\Psi\land\Xi\) that differ on some \(\phi\in\Phi_i\).
      \item If no such pair exists, the substate is \(\Phi_i\)-consistent → kernel \(K^{(i)}(S') = S'\).
    \end{itemize}
  \item \textbf{Admissibility}  
    Let \(I^* = \max\{i\mid S'\text{ is }\Phi_i\text{-consistent}\}\).  
    \begin{itemize}
      \item If \(I^*\) exists → \(\textsc{Admissible\_Level}(I^*)\); return \(I^*\) and the kernel formula \(\Psi\land\Xi\).
      \item Otherwise → \(\textsc{Refusal}\).
    \end{itemize}
  \item \textbf{Mode profile}  
    For each level \(i\):
    \begin{itemize}
      \item \textsc{Strict}: if \(\Psi\land\Xi\) has exactly one model (uniqueness check via solver).
      \item \textsc{Safe}: if \(S'\) is \(\Phi_i\)-consistent and has at least two models.
      \item \textsc{Defeasible}: otherwise.
    \end{itemize}
    The profile is descriptive only; admissibility is determined solely by \(I^*\).
  \item \textbf{Solver interaction}  
    \begin{itemize}
      \item All logical checks are implemented via SMT solver calls.
      \item If a query returns \(\textsc{Unknown}\) (e.g., due to undecidability or timeout) → \(\textsc{Refusal}\).
      \item All formulas are restricted to a decidable logic (e.g., QF\_LIA, QF\_UF).
    \end{itemize}
\end{enumerate}

This execution semantics guarantees that the framework is \textbf{mechanically executable} and \textbf{deterministic}, adhering to the fail‑closed principle.

\section{Conclusion}
\label{sec:conclusion}

\subsection{Core Insights}
Epistemic logic is replaced by constraint‑based state evolution. Knowledge is not belief, but invariant set under admissible transitions. Uncertainty is not probability, but emptiness at higher invariant levels. Reasoning is restriction, not generation.

\subsection{System Properties}
\begin{itemize}
  \item Fail‑closed
  \item No implicit assumptions
  \item Existence equivalent to admissibility
  \item Knowledge formation tied to level‑relative kernels \(K^{(i)}\)
  \item Controlled branching through Strict, Safe, and Defeasible modes per level
  \item Knowledge gradation via invariant hierarchy
  \item Multi‑agent constraints for communication, consistency, and privacy
  \item Paradox prevention within admissible state space
  \item Topological grounding: the admissible state space carries an Alexandrov topology in which kernel sets are open
  \item Relational admissibility: the two sequential gates RGR and RCR define a complete pipeline
  \item Dynamic updates: knowledge sharpens (never coarsens) via restriction of state space and reachability
  \item Formal theorem collection: all core properties are proved from the axioms
  \item Categorical extension: a thin category of reachability collapses Yoneda equivalence to identity, motivating enriched morphisms for non‑trivial semantic pairs
\end{itemize}

\subsection{Final Statement}
The system does not solve classical paradoxes. It prevents their construction within the admissible state space.

\subsection{Core Principles}
\begin{quote}
Truth is what survives all admissible transformations.\\
Reasoning is restriction, not generation.\\
Ambiguity is structural failure.\\
A system is reliable by what it refuses.\\
No interpretation. No implicit meaning. No assumptions.\\
Only: definition \(\rightarrow\) rule \(\rightarrow\) consequence
\end{quote}
